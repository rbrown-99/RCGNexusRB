# Brand assets

Place the official Albertsons logo here as one of:

- `albertsons-logo.svg` (preferred — vector, scales cleanly)
- `albertsons-logo.png` (fallback — use a transparent-background PNG, ~600px wide)

`BrandLogo.tsx` will pick it up automatically at `/albertsons-logo.svg` or
`/albertsons-logo.png`. If neither file is present, the component falls back
to a generic placeholder mark so the build still works.
